PROYECTO AXIE INFINITY BECAZUELA

Es el proyecto de un youtuber mexicano llamado BuXXX que tiene el objetivo de becar a la mayor cantidad de Venezolanos posible para mejorar su situacion economica drasticamente. Para obtener una beca solo debes ser activo y aportar valor. Cabe destacar que Bux renuncio a la idea de comprarse un Tesla Model X con el objetivo de ayudar a gran cantidad de venezolanos a salir de la pobreza mediante este proyecto.
